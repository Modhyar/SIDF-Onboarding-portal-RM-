Project Finance — Appraisal & Examples

For project finance, the RM officer is responsible for preparing the appraisal, which focuses primarily on the project’s feasibility, sponsor strength, long-term cash-flow generation, and repayment capacity. The appraisal includes the following:

1) Ownership Assessment Report

  • Ownership structure, shareholders and company’s management structure
  • Financial analysis for the borrower and related parties
  • Net worth assessment
  • Credit check and SIDF relationship

2) Project Evaluation Report

  • Project background, scope and rationale
  • Ownership assessment
  • Products and capacities
  • Project implementation status
  • Market Section
  • Financial analysis summary
  • Project costs and sources of finance
  • Loan section
  • Final recommendation

3) Financial model

For project finance loan, RM develops two financial perspectives: a standalone model to assess the project itself and a consolidated model to assess the sponsor’s overall financial strength.
